Karmac Dashboard

Project Specification v0.3

Everything you need. Nothing you don't.

What is Karmac?

Karmac is a free and open source desktop dashboard application for Linux. It gives everyday users a single, beautiful, organized view of their system's health and key personal information — without requiring any technical knowledge to set up or use.

Karmac is built on a simple belief: Linux desktop users deserve a polished, privacy-respecting dashboard that works for everyone, not just developers and power users.

Who is Karmac For?

  • Everyday users who are new to Linux and want something that just works
  • Privacy-conscious users switching away from proprietary operating systems
  • Experienced Linux users who want a cleaner, more unified dashboard experience
  • Gamers and enthusiasts who want real-time hardware monitoring in a friendly interface

Core Philosophy

  • Privacy first — Karmac collects no user data, sends nothing to external servers, and never will
  • Free and open source — licensed under GPL v3, free forever with no premium tier
  • Approachable — designed for humans, not just power users
  • Honest — what you see is what you get; no hidden features, no upsells, no ads
  • Community driven — built in the open, with contributions welcomed from anyone

Dashboard Panels (v2.0)

Row 1 — Personal and Connectivity

Clock and Calendar

Time, date, and mini calendar with today highlighted in blue.

Weather

Current conditions, high/low, humidity, wind speed, and 3-day forecast.

Network

Live upload/download speeds, ping/latency with quality rating, and internet speed test.

Row 2 — Thermal and Processing

Temperature

CPU and GPU temperatures in Celsius, Fahrenheit, or both. Color-coded thresholds.

Fan Speeds

RPM readings grouped by hardware chip. Zero RPM detection for GPU fans.

CPU Cores

Overall usage, per-core activity percentages, and per-core frequencies in GHz.

Row 3 — Memory and Storage

System Uptime

Time since last boot with boot timestamp.

RAM Usage

Used and available memory with color-coded percentage.

Hard Drives

Space used and free for all detected drives. Individual drives toggleable in settings.

Row 4 — Power and Graphics

GPU Usage

AMD GPU load percentage, VRAM usage, and power draw in watts.

Power Usage

CPU package and GPU power consumption in watts with combined total.

FPS

Live FPS, frametime, CPU/GPU load via MangoHud integration. Shows active game name.

Sidebar

The navigation sidebar displays full hardware information alongside the Dashboard and Settings navigation buttons. Hardware details include CPU, GPU, motherboard, OS, kernel version, desktop environment, RAM with speed, storage, and display information with resolution and refresh rate.

Settings

  • Appearance — Theme (Dark/Light), Font Size (Small/Medium/Large)
  • Panels — Enable/disable individual panels
  • Clock — 12/24h format, seconds, date format, timezone
  • Weather — Location search, temperature units
  • Network — Ping host configuration
  • Fan Speeds — Warning/critical RPM thresholds, custom fan labels
  • Temperature — Display format, warning/critical thresholds
  • Hard Drives — Show/hide individual drives
  • Startup — Launch on login

Technical Notes

  • Standalone desktop application targeting Linux
  • Written in Python with PySide6 (Qt framework)
  • No external server or hosting dependency
  • Weather data from Open-Meteo (free, no API key required)
  • FPS data from MangoHud log files
  • Power monitoring via RAPL interface
  • Crash logging to ~/.config/karmac/karmac.log
  • Licensed under GNU General Public License v3 (GPL v3)
